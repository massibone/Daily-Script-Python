
# Data Privacy Tools

Questa cartella contiene script Python per la gestione e l'anonimizzazione di dati sensibili, come feedback dei clienti, log o informazioni personali. Gli script sono progettati per aiutare a conformarsi alle normative sulla privacy (es. GDPR) e per proteggere l'identità degli utenti durante l'analisi dei dati.
